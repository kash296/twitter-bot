# twitter-bot
  A twitter bot by Kaushik Murali.  
  
  This bot will search Wikipedia for the current date and tweets about the events that took place on this day over the years.  
  
  Goals:
  - Perform Continuous Integration using Jenkins - Done
  - Perform Continous Deployment using docker - Done 
  - Try and automate it using an ansible playbook - Done
